# Créditos - OPNscript

OPNscript es posible gracias a la contribución de muchas personas y proyectos de código abierto. Esta página reconoce el trabajo de todos aquellos que han hecho que OPNscript sea una realidad.

## Creadores Principales

### **Aminá Ben (aminbena010-ai)**
- **Rol**: Creador y Desarrollador Principal de OPNscript
- **Contribuciones**: Diseño del lenguaje, transpilador, API de gráficos, editores y herramientas
- **Repositorio**: [GitHub - OPNscript](https://github.com/aminbena010-ai/OPNscript)

## Tecnologías Principales

### **Python 3.8+**
- **Creador**: Guido van Rossum y la comunidad Python
- **Licencia**: Python Software Foundation License
- **Uso**: Lenguaje base para el transpilador y runtime de OPNscript
- **Sitio Web**: [python.org](https://www.python.org/)
- **Créditos**: La comunidad Python ha proporcionado un lenguaje robusto y versátil que es la base de OPNscript.

### **Tkinter**
- **Creador**: Creado por Fredrik Lundh con contribuciones de la comunidad Python
- **Licencia**: Python Software Foundation License (parte de Python)
- **Uso**: Motor de renderizado de gráficos 2D integrado en OPNscript
- **Sitio Web**: [tkinter.tcl.tk](https://www.tcl.tk/)
- **Créditos**: Tkinter proporciona la funcionalidad gráfica que hace posible el módulo `gfx` de OPNscript.

### **PyQt6**
- **Creador**: Riverbank Computing Limited
- **Licencia**: GNU General Public License v3.0 (GPL v3) / Comercial
- **Uso**: Framework para el editor avanzado de escritorio tipo VS Code
- **Sitio Web**: [riverbankcomputing.com/software/pyqt](https://www.riverbankcomputing.com/software/pyqt/)
- **Créditos**: PyQt6 proporciona los componentes GUI avanzados necesarios para el editor de escritorio interactivo.

## Inspiración y Influencias

OPNscript está inspirado en la sintaxis y características de:

- **C/C++** - Sintaxis clara y operadores fundamentales
- **Python** - Legibilidad y potencia
- **JavaScript** - Funciones de primera clase y flexibilidad
- **Lua** - Simplicidad y embedded scripting
- **Processing** - Accesibilidad a gráficos y visualización
- **GLSL** - Conceptos de renderizado y shaders

## Cómo Contribuir

Si deseas contribuir a OPNscript y ser reconocido:

2. Realiza tus cambios en una rama nueva
3. **Envía un Pull Request** con descripción detallada
